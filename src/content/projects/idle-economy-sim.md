## About the Project

Idle games often suffer from poorly balanced economy that leads to player frustration or too quick completion.

### Solution

Idle Economy Simulator is an experimental project focused on testing various economic models and their impact on gameplay experience. It serves as a testing ground for game design concepts.

### Features

- Economic system simulation
- Testing various balancing models
- Game data visualization
- Statistics export for analysis

The game is available on itch.io and actively updated.
